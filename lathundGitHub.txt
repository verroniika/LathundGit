Steg för att ladda upp en mapp till GitHub
1. Gå in i mappen (git bash i mappen eller cb 'mappen')
2. git init
3. git status
4. git add 'fil' (eller -A om man vill adda alla filer i mappen)
5. git commit -m "Meddelande om ändringar eller vad man gjort med filen"
   Add rep på GitHub, följ instruktionerna (förslag 2)
6. För att pusha igen, gör steg 3-5 och skriv sen 'git push'
