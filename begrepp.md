Standard
-----------

CRUD:
`Create Read Update Delete`

Adda alla filerna för att sen commita:
`git add -A`

Adda speciel fil för att sen commita:
`git add index.html`

Committa ändringar:
`git commit -m "Meddelande"`

Add och commit i ett steg:
`git commit -am "Meddelande"`

Radera filer från Git:
`git rm index.html`
